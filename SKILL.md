# LinkedIn Management

Use the `browser` tool to navigate and interact with LinkedIn. This skill provides guidelines for professional LinkedIn profile management and content posting.

## Getting Started

1. Open LinkedIn: `https://www.linkedin.com/`
2. Navigate to profile or feed as needed
3. Use browser actions to read, edit, and post

## Profile Management

### Viewing a Profile
- Navigate to `https://www.linkedin.com/in/{username}/`
- Use snapshot to view full profile
- Key sections: About, Experience, Skills, Education

### Editing Headline
1. Click profile photo → "Edit intro"
2. Update headline field
3. Save changes

### Editing About Section
1. Go to profile → click "About" → "Edit"
2. Write compelling professional summary
3. Save

### Adding Skills
1. Go to profile → "Add skills"
2. Search and select relevant skills
3. Endorse others to receive endorsements

## Posting Content

### Creating a Post
1. Click "Start a post" on home feed
2. Write content in text field
3. Add hashtags, mentions, or media as needed
4. Click "Post" to publish

### Post Best Practices
- **Professional but personal**: Share insights, not just articles
- **Use hashtags**: 3-5 relevant tags (e.g., #AI, #Tech, #Career)
- **Call to action**: Ask questions, invite opinions
- **Keep it readable**: Short paragraphs, scannable format
- ** authenticity**: Show the human behind the profile

### Example Post Templates

Introduction post:
```
Hello, [Platform]! 👋

I'm [Name], [Title] at [Company]. Excited to connect with professionals in [Industry/Field].

[Brief personal angle or interest]

Let's connect and share insights!

#Industry #Professional #Introduction
```

Project/Work update:
```
Just shipped [Project/Achievement]! 🚀

[What it is in 1-2 sentences]

[What you learned / why it matters]

#ProjectName #Industry #Learning
```

## Engagement

### Liking Posts
- Click "Like" button on posts
- Hover for reaction options (celebrate, support, etc.)

### Commenting
- Click "Comment" → type response
- Add value, ask questions, or share related experience

### Sharing Posts
- Click "Repost" to share with your own commentary
- Add original thought to increase visibility

## Professional Branding Tips

### Profile Optimization
1. **Headline**: Include role + unique value proposition
2. **Photo**: Professional, friendly, clear face
3. **About**: Tell your story in ~3 paragraphs
4. **Experience**: Focus on impact and achievements
5. **Skills**: Relevant to your target roles

### Content Strategy
- Post 2-5x per week for visibility
- Mix: original thoughts, industry news, personal wins
- Engage with others' content to build network
- Be consistent but authentic

### Voice Guidelines
- Professional but approachable
- Share failures + learnings (more relatable)
- Avoid overly corporate language
- Show personality

## Common Tasks

| Task | URL/Action |
|------|-------------|
| View own profile | `linkedin.com/feed/` → click your name |
| Edit profile | `linkedin.com/in/{username}/` → "Edit" buttons |
| Post to feed | Home → "Start a post" |
| View notifications | Click bell icon |
| Search | Use search bar at top |

## Notes

- LinkedIn has rate limits; space out actions
- Browser session should be logged in manually first
- Some features require LinkedIn Premium
- Always review posts before publishing
